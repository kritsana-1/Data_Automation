# Automated Data Insights Dashboard

ระบบวิเคราะห์ข้อมูลอัตโนมัติที่ช่วยเปลี่ยนไฟล์ Excel/CSV ที่ซับซ้อน ให้กลายเป็นกราฟสรุปผลที่เข้าใจง่าย พร้อมระบบแจ้งเตือนผ่าน Line Notify

## ฟีเจอร์เด่น
- **Easy Upload:** รองรับทั้งไฟล์ .csv และ .xlsx (Excel)
- **Auto Analysis:** คำนวณสถิติพื้นฐานและยอดรวมให้อัตโนมัติ
- **Interactive Graphs:** กราฟปรับเปลี่ยนตามข้อมูลที่เลือกแบบ Real-time
- **Line Notification:** ส่งสรุปยอดสำคัญเข้ามือถือได้ทันทีผ่าน Line API
